Cara Menjalankan Program C++ di VS Code

Berikut langkah-langkah untuk menyiapkan dan menjalankan program C++ di Visual Studio Code pada Windows:

Langkah 1: Install Visual Studio Code

Download dan install Visual Studio Code dari tautan resmi berikut:

Download VS Code

Langkah 2: Install Compiler C++ MinGW

Selanjutnya, kamu perlu menginstall compiler MinGW untuk mengkompilasi dan menjalankan program C++. Ikuti langkah-langkah berikut:

  • Kunjungi MinGW-w64.

  • Klik `mingw-19.0.exe` untuk mendownload software.
  • Setelah selesai, install dan buat di direktori C. Kemudian klik extract.

  • Buka file explorer, masuk ke direktori C, lalu klik folder Mingw.

  • Masuk ke folder `bin`.

  • Salin lokasi folder `bin` seperti gambar berikut.

  • Buat environment variable di perangkat kamu, klik open.

  • Klik environment variables, lalu klik Path, dan pilih Edit.


  • Tambah yang baru, lalu paste folder `bin` yang tadi disalin.

Langkah 3: Siapkan C++ di VS Code

Setelah MinGW terinstall, ikuti langkah berikut untuk menyiapkan C++ di VS Code:

  • Buka VS Code dan install ekstensi **C/C++** dari marketplace ekstensi seperti di gambar berikut.

  • Buat file baru dengan ekstensi `.cpp` di VS Code.

  • Tulis program C++ sederhana (misalnya, "Hello World").

Langkah 4: Kompilasi dan Jalankan Program


Untuk menjalankan program, buka terminal di VS Code dan ketik perintah berikut:

g++ yourfile.cpp -o yourfile.exe
./yourfile.exe

Perintah tersebut akan mengkompilasi dan menjalankan program C++ kamu.
